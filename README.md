## Description

A simple Project created while learning nestjs, class-validator, and prisma.

## Installation
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# Create the db and run docker container, and run prisma migrate deploy
$ npm run db:dev:restart
```
